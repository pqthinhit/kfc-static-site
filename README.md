# 🍗 KFC Landing Page

## 📌 Giới thiệu

Bài tập nhỏ xây dựng trang web tĩnh lấy cảm hứng từ giao diện KFC (đã custom lại).

Mục tiêu là luyện tập **HTML, CSS** và **animation cơ bản**.

---

## 🎯 Nội dung học được

* Cấu trúc HTML cơ bản
* Styling với CSS (Flexbox)
* Animation cơ bản (hover, transition, keyframes)
* Tổ chức code đơn giản

---

## 🛠️ Công nghệ

* HTML5
* CSS3
* CSS Animation

---

## 🚀 Tính năng

* Giao diện phong cách KFC (custom)
* Hiệu ứng hover và animation nhẹ
